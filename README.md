### How to use

``` bash
pip install -r requirements.txt
git clone https://github.com/HouHou0925/JPEG.git
python main.py
```

輸入檔名 ex:Baboon.raw
輸入qf ex:50

直接產生解壓縮後的圖檔 (因為要比較大小所以存成.png)
